Preface

There aren't many good GIF editors. Online editors like Ezgif are extremely limited, as they only allow you to modify existing GIFs and not create new ones. Offline editors are usually way too advanced, like Photoshop or ImageReady. I wanted to make an editor for myself that would be both easy to use and very powerful. It would specifically focus on the task of creating GIFs, fast.

Description

GrrGif is a work-in-progress GIF editor for Win32-based systems. It strives to offer the most features while being slim and mindful of resources.
