J2ME Projects

Preface

J2ME (Java 2 Micro Edition) is a lightweight version of Java used on old feature phones in the 2000s from manufacturers such as Nokia or Samsung.
Having a J2ME phone myself, I found it useful to write some very simple apps for it.
I will not provide downloads for them for now, maybe some time in the future I will.

Projects

Test Network

Tool that lets me see if GPRS/EDGE connection is available, shows the public IP used and measures its speed.
